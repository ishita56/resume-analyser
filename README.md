
AI-Powered Resume Analyzer

An AI-driven web application that analyzes resumes against job descriptions, calculates ATS-style match scores, identifies skill gaps, and generates detailed feedback reports to help candidates improve their chances of landing interviews.

🚀 Live Demo

https://resume-analyser-ishita56s-projects.vercel.app

📖 About The Project

As a Computer Science student preparing for placements, I wanted to understand how modern ATS (Applicant Tracking Systems) and AI-powered resume screening tools work.

This project analyzes resumes against job descriptions using AI-based scoring and skill matching, helping users identify strengths, missing skills, and areas for improvement. It also generates detailed feedback reports that can be downloaded and reviewed later.

✨ Features
Upload resumes in PDF format
Automatic resume parsing and information extraction
Job description-based resume evaluation
ATS-style match score generation
Skill matching and missing skill detection
AI-generated feedback and recommendations
Downloadable PDF analysis reports

📂 Project Workflow
User uploads a resume.
User enters or pastes a job description.
Resume content is extracted and parsed.
AI analyzes the resume against the job description.
The system calculates:
ATS Match Score
Matched Skills
Missing Skills
Personalized Recommendations
A detailed analysis report is generated.
Users can download the report or view previous analyses.
